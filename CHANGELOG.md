# Changelog

本文档记录雪峰Agent桌面版的版本更新历史。

## [0.1.2] - 2026-06-25

### 新增

- **志愿填报模拟页面**：独立窗口，参考河南官方志愿草表格式（双行表头、每志愿两行），默认48个志愿，支持动态增减数量
- **专业代码选择器**：支持输入+下拉选择，内置1954个专业数据（普通本科833+专科801+职教本科320）
- **Excel 导入导出**：使用 Tauri 原生文件对话框，格式与官方草表一致
- **数据持久化**：志愿数据自动保存到 localStorage

### 改进

- **主题同步**：所有独立窗口页面支持主题同步切换
- **表格样式优化**：志愿填报表格添加外边距和边框，输入框动态撑满单元格宽度
- **志愿数量输入框**：支持直接输入志愿数量，失焦或回车后调整，替代逐个点击添加
- **表头样式**：添加主题色渐变背景，提升视觉层次感
- **表格对齐**：序号和是否调剂列垂直居中对齐

### 依赖

- 新增 `xlsx`（Excel 读写）
- 新增 `@tauri-apps/plugin-dialog`（原生文件对话框）
- 新增 `@tauri-apps/plugin-fs`（文件系统读写）

## [0.1.1] - 2026-06-24

### 新增

- **自定义 Markdown 渲染组件**：替代 v-md-preview，修复表格深色模式/列宽/对齐问题，支持自定义主题样式
- **三路并发联网搜索**：验证 DB 学校（冲稳保各 5 所）、补全年度数据、行业趋势就业，对齐原项目搜索逻辑
- **百度搜索兜底**：Tavily 不可用时自动降级到百度搜索（Rust 后端）
- **系统托盘菜单**：显示窗口 / 隐藏窗口 / 退出（参考喵咪麦克），关闭窗口隐藏到托盘而非退出
- **多窗口架构**：独立设置窗口（API 设置、软件设置），通过 Tauri 事件通信
- **服务商预设**：内置 DeepSeek、小米 MiMo、通义千问、智谱 GLM 等 9 家服务商配置
- **主题系统**：CSS 变量驱动，浅色奶油白+橘色 / 深色黑色+绿色（参考喵咪麦克配色）
- **右键删除消息**：右键任意消息可删除，用户消息连带 AI 回复一起删
- **在线更新**：基于 Tauri Updater + GitHub Releases，支持检查更新、下载、静默安装
- **更新签名密钥**：NSIS 安装包支持签名验证
- **应用图标替换**：使用自定义应用图标，替换 Tauri 默认图标
- **数据库自动解压**：首次启动自动解压 admission_clean.db.gz（28MB，覆盖 8 省 26 万条数据）

### 改进

- **版本号动态获取**：Sidebar 和软件设置页面版本号改为 `getVersion()` 动态获取
- **NSIS 安装器中文语言**：安装界面支持简体中文
- **URL 自动处理**：自动处理 `/v1` 重复拼接（如小米 MiMo 的 URL 已含 `/v1`）
- **消息历史截取**：最多保留最近 25 条消息发送给 LLM，避免 token 超限

### 文档

- 更新 CLAUDE.md、README.md、版权声明
- 添加服务商预设配置说明
- 添加数据库覆盖省份说明

## [0.1.0] - 2026-06-23

### 新增

- 项目初始化，基于 xuefeng-agent 项目改造为 Tauri 2 桌面应用
- 基础聊天功能：多轮对话、新建/切换/删除对话
- 双模式切换：报考模式（正式志愿分析）←→ 娱乐模式（张雪峰风格）
- AI 智能提取：自动从描述中识别省份、位次、专业偏好
- 本地数据库查询：Rust 后端 SQLite 查询，冲稳保三级推荐
- 联网搜索：Tavily 搜索集成
- 深色/浅色主题切换
- API 设置：支持多家 AI 服务商
