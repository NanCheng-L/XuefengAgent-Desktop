# 雪峰Agent桌面版 — AI 高考志愿顾问

> 基于 [xuefeng-agent](https://github.com/NanCheng-L/xuefeng-agent) 改造的桌面应用版本。
> 内置 24 省官方录取数据库，真的会查数据、盘志愿。

## 功能特性

- **本地数据库**: 24 省 × 2024-2025 年官方投档线，42 万条，位次精确匹配
- **AI 智能提取**: 自动从描述中识别省份、位次、专业偏好
- **联网搜索**: Tavily 优先，百度兜底，搜索最新分数线和就业信息
- **多轮对话**: 上下文记忆，追问不用重复说
- **深色模式**: 晚上用不刺眼
- **隐私安全**: 数据全在本地，Key 存本地，不上传服务器

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **桌面框架**: Tauri 2.0
- **后端**: Rust (Tauri 2)

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 启动开发模式

```bash
# 仅前端
pnpm dev

# 完整桌面应用
pnpm tauri dev
```

### 构建生产版本

```bash
pnpm tauri build
```

## 使用方法

1. 启动应用后，点击右上角「API设置」
2. 填写 DeepSeek API Key（或其他 OpenAI 兼容 API）
3. 可选填写 Tavily Key 以启用联网搜索
4. 在输入框输入省份、分数、位次等信息
5. 获取冲稳保志愿推荐

## 示例输入

```
浙江655分位次10500，想学计算机电子，不接受调剂
```

## 支持的 AI 模型

任何 OpenAI 兼容 API 都能用：

| 模型 | 推荐 | 说明 |
|------|------|------|
| DeepSeek | ⭐首选 | 便宜好用，送免费额度 |
| 通义千问 | 推荐 | 有免费额度 |
| 智谱 GLM | 推荐 | 国产稳定 |
| GPT-4o | 可选 | 贵，需网络 |
